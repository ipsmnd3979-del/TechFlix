Create a complete responsive OTT Streaming Platform named **TechFlix** using

* MySQL Database
* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* AJAX
* JSON APIs
* Font Awesome
* Chart.js
* SweetAlert2

The website should look modern like Netflix with a black-red theme.
